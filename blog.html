Importent FAQ ?

1. What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the page structure as a tree of objects.

2. How do you select an element from the DOM?

You can use methods like `document.querySelector()` or `document.getElementById()`.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows you to handle events at a higher level in the DOM. It is useful for handling events on dynamically added elements.

4. How do you manipulate an element's attributes and styles using the DOM?

You can use `element.style`, `setAttribute()`, and `classList` methods to change attributes and styles.
